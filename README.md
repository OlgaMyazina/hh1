# hh1
task 1 for school.hh.ru:
1. Точки
Даны N точек на плоскости (для простоты можно считать, что у всех точек целочисленные координаты). Назовём расстояние от точки A до ближайшей к ней точки B "радиусом" точки A. "Соседями" точки A будем называть все точки, лежащие в пределах двойного радиуса от неё включительно (кроме самой точки A).
 
Для каждой точки из заданного набора определите её радиус и количество соседей.
===========================

Программа решает задачу Точки. При запуске запрашивает желание ввести координаты точки на плоскости, для подтверждения намерения необходимо ввести символ Y. Если все точки введены, то нужно ввести символ N.
После проверки на корректность ввода данных, программа для каждой точки рассчитает радиус и количество соседей и выведет эту информацию на экран. Для завершения программы - ожидает ввод любых данных.
